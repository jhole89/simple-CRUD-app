Simple CRUD

A simple CRUD web app example

Usage example:

  • Create - POST: /contacts/create body/JSON: { "username": "simpo", "email": "chunkylover52@aol.com", "first_name": "homer", "surname": "simpson" } - creates a new contact "simpo"
  • Read - GET: /contacts/simpo - returns details for the user "simpo", or all users if none provided
  • Update - POST: /contacts/update body/JSON: { "username": "simpo", "email": "homer.simpson@gmail.com", "first_name": "Homer", "surname": "Simpson" } - updates the contact "simpo" with new details
  • Delete - DELETE: /contacts/delete/simpo - deletes the contact "simpo"
